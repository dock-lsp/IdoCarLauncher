## 主界面功能

### 界面布局

主界面采用车载优化的横屏布局设计：

```
┌─────────────────────────────────────────────────────────────┐
│  [时间]    [日期]                              [设置] [主题] │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
│  │  应用1   │  │  应用2   │  │  应用3   │  │  应用4   │        │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘        │
│                                                             │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
│  │  应用5   │  │  应用6   │  │  应用7   │  │  应用8   │        │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘        │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  [导航卡片]              │  [媒体卡片]                      │
│  目的地 / 导航信息        │  歌曲名 - 艺术家                 │
├─────────────────────────────────────────────────────────────┤
│  [快捷方式1] [快捷方式2] [快捷方式3] ...                    │
└─────────────────────────────────────────────────────────────┘
```

### 功能特性

#### 1. 应用网格
- **4列网格布局**：适合车载横屏显示
- **应用图标显示**：显示应用名称和图标
- **长按菜单**：长按应用图标显示操作菜单（收藏、卸载、信息）
- **智能排序**：支持按名称、使用频率、安装时间排序

#### 2. 快捷方式栏
- **水平滚动**：支持添加多个快捷方式
- **快捷类型**：
  - 应用快捷方式：直接启动应用
  - 功能快捷方式：快速打开设置项（WiFi、蓝牙、音量等）
  - 小部件快捷方式：显示实时信息

#### 3. 信息卡片
- **导航卡片**：显示当前导航状态或导航入口
- **媒体卡片**：显示当前播放的音乐信息和控制按钮
- **车辆信息**：显示车速、温度等车辆信息（需车辆数据支持）

#### 4. 顶部工具栏
- **时间日期**：实时显示当前时间和日期
- **设置入口**：快速进入设置界面
- **主题切换**：快速切换主题
- **语音助手**：启动语音助手

## 导航系统

### 功能概述

导航系统提供完整的车载导航体验，支持画中画模式和多种导航应用。

### 核心功能

#### 1. 画中画导航 (PiP)
- **悬浮显示**：导航信息以小窗口形式悬浮显示
- **实时更新**：显示剩余距离、预计到达时间
- **路线指引**：显示当前道路和下一个转向提示
- **一键展开**：点击画中画窗口可全屏显示

#### 2. 多导航应用支持
自动检测并支持以下导航应用：

| 应用名称 | 包名 | 支持状态 |
|----------|------|----------|
| 高德地图 | `com.autonavi.minimap` | ✅ 完全支持 |
| 百度地图 | `com.baidu.BaiduMap` | ✅ 完全支持 |
| Google Maps | `com.google.android.apps.maps` | ✅ 完全支持 |

#### 3. 位置服务
- **GPS定位**：高精度卫星定位
- **网络定位**：WiFi/基站辅助定位
- **位置监听**：1秒更新频率，5米移动阈值

### 使用流程

```
1. 点击主界面导航卡片
2. 选择目的地输入方式：
   - 语音输入："导航到XX"
   - 手动输入：在搜索框输入地址
   - 历史记录：选择常用地址
3. 选择导航应用（如已安装多个）
4. 开始导航
5. 返回主界面，导航信息以画中画形式显示
```

## 媒体控制

### 功能概述

媒体控制模块统一管理车载环境中的音乐播放，支持多种音频源。

### 支持的媒体源

| 类型 | 说明 | 示例应用 |
|------|------|----------|
| 蓝牙音乐 | 手机蓝牙音频 | 系统蓝牙 |
| 在线音乐 | 流媒体音乐应用 | 网易云音乐、QQ音乐、Spotify |
| 本地音乐 | 本地存储音频 | 系统音乐播放器 |

### 核心功能

#### 1. 播放控制
- **播放/暂停**：控制音乐播放状态
- **上一首/下一首**：切换曲目
- **进度调节**：拖动进度条调整播放位置
- **音量控制**：独立音量调节

#### 2. 播放模式
- **顺序播放**：按列表顺序播放
- **随机播放**：随机选择曲目
- **单曲循环**：重复播放当前曲目
- **列表循环**：循环播放整个列表

#### 3. 播放信息显示
- **歌曲名称**：当前播放曲目
- **艺术家**：演唱者信息
- **专辑封面**：显示专辑图片
- **播放进度**：当前位置/总时长

### 媒体播放器界面

## 语音助手

### 功能概述

智能语音助手提供免手操作的交互方式，让驾驶更安全。

### 支持的语音命令

#### 1. 打开应用
```
语音指令："打开XX应用" / "启动XX"
示例："打开导航" / "启动音乐"
```

#### 2. 播放音乐
```
语音指令："播放音乐" / "播放XX的歌" / "播放XX"
示例："播放周杰伦的歌" / "播放晴天"
```

#### 3. 导航
```
语音指令："导航到XX" / "去XX怎么走" / "带我去XX"
示例："导航到北京天安门" / "去最近的加油站"
```

#### 4. 拨打电话
```
语音指令："打电话给XX" / "呼叫XX" / "拨打XX"
示例："打电话给张三"
```

#### 5. 系统设置
```
语音指令："打开蓝牙" / "打开WiFi" / "调大音量"
示例："打开蓝牙" / "音量调到50%"
```

#### 6. 天气查询
```
语音指令："今天天气怎么样" / "明天会下雨吗"
示例："查询北京天气"
```

### 语音交互流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  唤醒助手  │ -> │  聆听语音  │ -> │  识别处理  │ -> │  执行反馈  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
     │               │               │               │
     ▼               ▼               ▼               ▼
  点击语音按钮    显示聆听动画    显示识别文字    语音播报结果
  或语音唤醒      等待用户说话    处理命令意图    执行对应操作
```

### 语音识别状态

| 状态 | 说明 | 界面显示 |
|------|------|----------|
| Idle | 空闲状态 | 等待唤醒 |
| Listening | 正在聆听 | 波形动画 |
| Processing | 处理中 | 转圈动画 |
| Recognized | 已识别 | 显示文字 |
| Completed | 执行完成 | 显示结果 |
| Error | 错误 | 错误提示 |

## 插件系统

### 功能概述

插件系统允许第三方开发者为 IdoCarLauncher 扩展功能，实现模块化架构。

### 插件类型

#### 1. 内置插件
- 随应用打包发布
- 位于 `assets/plugins/` 目录
- 通过 JSON 配置定义

#### 2. 外部插件
- 独立 APK 文件
- 用户自行安装
- 支持热插拔

### 插件结构

```
插件 APK 结构：
├── AndroidManifest.xml
│   └── <meta-data android:name="plugin_id" android:value="xxx"/>
│   └── <meta-data android:name="plugin_entry" android:value="xxx"/>
├── classes.dex
│   └── 插件入口类实现 CarLauncherPlugin 接口
└── resources.arsc
```

### 插件管理

#### 插件状态
- **已安装**：插件文件已存在
- **已启用**：插件正在运行
- **已禁用**：插件存在但未运行

#### 插件操作
- **安装**：从 APK 文件安装插件
- **启用/禁用**：控制插件运行状态
- **卸载**：删除插件文件
- **更新**：覆盖安装新版本

### 插件开发示例

```kotlin
// 1. 实现插件接口
class MyPlugin : CarLauncherPlugin {
    
    override fun onCreate(context: Context) {
        // 插件初始化
        Log.d("MyPlugin", "Plugin created")
    }
    
    override fun onDestroy() {
        // 插件销毁，清理资源
        Log.d("MyPlugin", "Plugin destroyed")
    }
    
    // 自定义功能方法
    fun doSomething() {
        // 实现功能
    }
}

// 2. AndroidManifest.xml 配置
<application>
    <meta-data 
        android:name="plugin_id" 
        android:value="com.example.myplugin" />
    <meta-data 
        android:name="plugin_entry" 
        android:value="com.example.myplugin.MyPlugin" />
    <meta-data 
        android:name="plugin_author" 
        android:value="Developer Name" />
</application>
```

## 主题系统

### 功能概述

主题系统允许用户自定义应用外观，支持多种预设主题和自定义主题。

### 内置主题

| 主题名称 | 特点 | 适用场景 |
|----------|------|----------|
| 默认主题 | 浅色背景，深色文字 | 白天使用 |
| 深色主题 | 深色背景，浅色文字 | 夜间使用，减少眩光 |
| 高对比度 | 高对比度配色 | 提高可读性 |

### 主题属性

主题可自定义的属性包括：

- **颜色**
  - 主色调 (Primary)
  - 次色调 (Secondary)
  - 背景色 (Background)
  - 表面色 (Surface)
  - 错误色 (Error)
  - 文字颜色 (On Primary/Secondary/Background)

- **字体**
  - 字体家族
  - 字号大小
  - 字重

- **圆角**
  - 卡片圆角
  - 按钮圆角
  - 输入框圆角

- **阴影**
  - 卡片阴影
  - 按钮阴影
  - 悬浮按钮阴影

### 主题配置文件

```xml
<!-- themes.xml -->
<style name="Theme.CarLauncher.Dark" parent="Theme.Material3.Dark.NoActionBar">
    <item name="colorPrimary">@color/dark_primary</item>
    <item name="colorOnPrimary">@color/dark_on_primary</item>
    <item name="colorPrimaryContainer">@color/dark_primary_container</item>
    <item name="colorSecondary">@color/dark_secondary</item>
    <item name="android:colorBackground">@color/dark_background</item>
    <item name="colorSurface">@color/dark_surface</item>
</style>
```

## 快捷操作

### 悬浮球

悬浮球提供随时可访问的快捷操作入口。

#### 功能特性
- **可拖拽**：可拖动到屏幕任意位置
- **点击展开**：点击展开快捷菜单
- **自动吸附**：靠近边缘时自动吸附

#### 快捷菜单
展开后显示以下快捷入口：
- 🏠 主页：返回主界面
- 🧭 导航：打开导航
- 🎙️ 语音：启动语音助手
- ⚙️ 设置：打开设置
- ❌ 关闭：关闭菜单

#### 使用方式
```
1. 点击悬浮球展开菜单
2. 选择需要的功能
3. 菜单自动关闭
4. 可拖动悬浮球调整位置
```

### 快捷方式

#### 添加快捷方式
1. 长按应用图标
2. 选择"添加到快捷方式"
3. 快捷方式出现在主界面底部

#### 快捷方式类型
- **应用快捷方式**：直接启动应用
- **功能快捷方式**：快速执行系统功能
  - WiFi 开关
  - 蓝牙开关
  - 音量调节
  - 显示设置
- **小部件快捷方式**：显示实时信息的小部件

## 设置中心

### 设置分类

#### 1. 通用设置
- **开机启动**：设备启动时自动运行
- **默认桌面**：设置为系统默认桌面
- **语言选择**：切换界面语言

#### 2. 显示设置
- **主题选择**：切换应用主题
- **亮度调节**：屏幕亮度设置
- **字体大小**：界面字体大小

- **媒体音量**：音乐播放音量

- **语音识别语言**：设置识别语言
- **TTS语音**：选择语音合成声音

#### 6. 插件管理
- **已安装插件**：查看和管理已安装插件
- **安装新插件**：从文件安装插件
- **插件市场**：浏览和下载插件

#### 7. 关于
- **版本信息**：显示当前版本号
- **检查更新**：检查新版本
- **开源许可**：查看开源组件许可
- **隐私政策**：查看隐私政策
